# ActiveMap Reviewer

Инструмент для массовой проверки заданий в системе ActiveMap (sao.geofsm.ru).  
Разработан для Префектуры САО.

## Что делает

- Загружает задания по выбранному виду работ и этапу (например, «выполнено» + «В работе»)
- Показывает карточку задания: фотографии, описание, кастомные поля, адрес, исполнитель
- Позволяет нажать кнопку решения (Утверждено / Доработка / другие) — статус сразу меняется в системе
- Работает полностью локально: не требует интернета кроме подключения к серверу ActiveMap
- Несколько сотрудников могут работать параллельно — каждый видит свежий список при обновлении

## Установка

### Вариант 1 — Python (для тех, у кого он установлен)

1. Скачайте `reviewer.py` и `reviewer_html.html` в одну папку
2. Откройте командную строку в этой папке
3. Запустите: `python reviewer.py`
4. Браузер откроется автоматически на `http://localhost:8765`

### Вариант 2 — готовый .exe (не требует Python)

1. Скачайте `ActiveMapReviewer.exe` из [Releases](../../releases)
2. Запустите файл — браузер откроется автоматически

## Использование

1. Введите логин и пароль от ActiveMap
2. В настройках выберите:
   - Вид задания (например, АБП)
   - Этап (статус в системе, например «выполнено»)
   - Стадия карточки (В работе / Завершено)
   - Кнопки решений (настройте метки и статусы для каждой кнопки)
3. Нажмите «Начать проверку»
4. Просматривайте задания, нажимайте кнопки или клавиши:
   - `1`, `2`, `3`... — кнопки решений (с автопереходом вперёд)
   - `S` / `Ы` — пропустить задание
   - `←` / `→` — навигация

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `1`–`9` | Кнопка решения (автопереход) |
| `S` / `Ы` | Пропустить |
| `←` `→` | Навигация |
| `Escape` | Закрыть лайтбокс |

## Файлы

| Файл | Назначение |
|------|-----------|
| `reviewer.py` | Серверная часть (CORS-прокси) |
| `reviewer_html.html` | Интерфейс |
| `bug_reports.jsonl` | Отчёты об ошибках (создаётся автоматически) |

## Сборка .exe

```bash
pip install pyinstaller
pyinstaller --onefile --noconsole --name ActiveMapReviewer reviewer.py
```

Файл появится в папке `dist/`.  
Рядом с `.exe` должен лежать `reviewer_html.html`.
