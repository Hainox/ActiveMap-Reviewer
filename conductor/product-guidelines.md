# Product Guidelines

## Brand Identity

**Продукт:** ActiveMap Reviewer
**Организация:** Префектура Северного административного округа г. Москвы, Управление ЖКХ
**Аудитория:** Государственные служащие, работающие с внутренними системами

### Tone of Voice

- **Официально, но не бюрократично:** Русский язык, профессиональный тон
- **Лаконично:** Минимум слов в UI-элементах; «Войти», а не «Нажмите для входа в систему»
- **Без ambiguity:** Статусы и действия названы точно — «Завершено», «В работе», «Отклонено»
- **Ошибки информативны:** «Нет соединения с сервером» вместо «Ошибка 503»

### Typography

- **Шрифт:** Системный стек — `-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif`
- **Размеры:** 14px (основной текст), 12px (метки/бейджи), 18px (заголовки карточек)
- **Вес:** 400 (обычный), 600 (кнопки, заголовки)

### Layout

**Экран входа** — двухпанельный:
- Левая панель (40%): тёмный фон `#1a1a2e`, герб Префектуры САО, название продукта, версия
- Правая панель (60%): белый фон, форма авторизации

**Рабочий экран** — трёхзонный:
- Топбар: тёмный `#1a1a2e` с золотым акцентом, логотип, имя пользователя, кнопка выхода
- Основная зона: карточка задания с фото и кастомными полями
- Нижняя зона: кнопки решений + история решений сессии

**Экран настройки** — одноколоночный со смысловыми блоками (`setup-block`).

## UI Component Rules

### Кнопки решений
- Цветные, min-height 48px
- Текст белый на любом фоне кнопки
- При нажатии: визуальный feedback, блокировка до получения ответа сервера
- Иконка статуса (⏳ / ✓ / ✗) рядом с названием

### Статусные бейджи
- Этап задания: скруглённый бейдж с цветом статуса
- Стадия (В работе / Завершено): отдельный бейдж
- Организация: серый бейдж

### Фотографии
- Сетка превью с lazy loading
- Лайтбокс: тёмный overlay, навигация стрелками и клавиатурой
- Зум дискретный: 1× → 1.25× → 1.5× → 2× → 2.5× → 3× → 4× → 5× → 6× → 8×

### Ошибки и состояния
- Ошибка загрузки: карточка с иконкой ✗ и кнопкой «Повторить»
- Пустой список: информативный текст
- Загрузка: спиннер или прогресс-бар, не белый экран

### Горячие клавиши
- Подсказка в интерфейсе: строка внизу с перечнем
- 1–5: кнопки решений, ←/→: навигация, S/Ы: пропуск, +/-: зум, 0: сброс, Esc: закрыть лайтбокс

## Quick Reference

1. Топбар всегда тёмный (#1a1a2e) с золотым акцентом (#c8a96a)
2. Русские тексты, официальный тон, без сокращений в важных элементах
3. Кнопки решений всегда цветные и крупные — основной инструмент оператора
4. Ошибки всегда видны пользователю — никаких молчаливых сбоев
5. Фон приложения светлый (#f5f5f5), карточки белые
6. Все состояния (загрузка / успех / ошибка) визуально различимы
