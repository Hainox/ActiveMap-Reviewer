# Product Definition

## Vision

Десктопный инструмент для сотрудников Префектуры САО, который превращает ежедневную ручную проверку сотен заданий в ActiveMap в быстрый поток: одна горячая клавиша — следующее задание.

## Problem Statement

Сотрудники Управления ЖКХ Префектуры САО ежедневно проверяют сотни заданий в ActiveMap: открывают каждое вручную, изучают фотографии, переключают статус. Процесс монотонный, медленный и не масштабируется — нет возможности работать потоком, нет горячих клавиш, нет фильтрации по нескольким критериям одновременно.

## Target Users

**Оператор Префектуры САО** — сотрудник отдела ЖКХ, который:
- Работает на Windows-ПК в локальной сети Префектуры
- Не имеет технических навыков (не устанавливает ПО, не знает командной строки)
- Ежедневно обрабатывает от 50 до 500+ заданий
- Авторизуется в системе ActiveMap (sao.geofsm.ru) по логину/паролю

## Success Criteria

- Оператор обрабатывает в 3–5× больше заданий за смену по сравнению с ручной работой в ActiveMap
- Настройки сохраняются между сессиями — не нужно заново вводить фильтры каждый запуск
- Статус задания обновляется с первой попытки; сбои видны оператору, не теряются молча
- Приложение запускается без установки: скачал .exe — запустил — работает

## Core Features

### Авторизация
- Логин/пароль через ActiveMap API
- Автодетект формата токена (Bearer, Token, Cookie, query-param, X-Auth-Token)
- Сохранение сессии в пределах одного запуска

### Фильтрация и настройка
- Выбор вида работы, этапа (статуса), стадии, организации, периода дат
- **v2: Сохранение настроек в localStorage** — фильтры и кнопки решений сохраняются между запусками
- **v2: Полная загрузка организаций** без ограничения 500 записей (рекурсивная пагинация)
- **v2: Индикатор загрузки** при получении списка организаций

### Просмотр заданий
- Последовательный просмотр заданий одно за другим
- Фотографии с лайтбоксом (зум до 800%, панорамирование, навигация)
- Кастомные поля с читаемыми русскими названиями
- **v2: Автоматическая разбивка заданий** на страницы вместо предупреждения «500 заданий»

### Принятие решений
- До 5 настраиваемых кнопок с именем, цветом, статусом (этапом) и стадией (В работе/Завершено)
- Горячие клавиши 1–9 для кнопок решений
- Горячие клавиши ←/→ навигация, S/Ы пропустить, +/- зум фото, 0 сброс зума, Escape
- **v2: Блокировка кнопок** пока идёт отправка PATCH — исключает дублирование

### Отправка статусов
- Фоновая очередь PATCH-запросов с задержкой 400мс
- **v2: Автоповтор при ошибке** с визуальным индикатором результата (ОК/ошибка)
- **v2: Логирование PATCH-запросов** в файл рядом с .exe

### История и итоги
- История решений текущей сессии с переходом по клику
- Экран итогов с количеством обработанных заданий
- **v2: Экспорт результатов сессии** в CSV/Excel

### Баг-репорты
- Форма 🐛 с сохранением в bug_reports.jsonl
- Отправка в GitHub Issues (если настроен токен)

## Non-Goals

- Мобильная версия (приложение для Windows-десктопа)
- Редактирование содержимого заданий (только смена статуса/стадии)
- Создание новых заданий в ActiveMap
- Авторизация без учётных данных ActiveMap (OAuth/SSO не планируется)
- Работа с несколькими организациями одновременно (одна сессия — один аккаунт)
- Онлайн-дашборд или веб-версия

## Constraints

- **Zero-install:** один `.exe` без зависимостей, PyInstaller + Python stdlib
- **Windows только:** целевая платформа Windows 10/11
- **Локальная сеть:** работа через https://sao.geofsm.ru, нет внешнего интернета на рабочих ПК
- **Нет Code Signing:** предупреждение SmartScreen при первом запуске — допустимо для внутреннего инструмента
- **Нет серверной части:** всё в одном .exe, нет базы данных, нет облака

## Initial Concept

v1.x — рабочая версия: auth, фильтры, просмотр заданий с фото, кнопки решений, горячие клавиши, баг-репорты, стадия в кнопках, полная загрузка организаций.

v2.0 — полная пересборка с нуля: устранение накопленного технического долга (глобальный стейт, молчаливые ошибки, 1782-строчный монолит) при сохранении всех функций. Добавление: localStorage-настройки, блокировка кнопок при отправке, логирование, экспорт CSV.
